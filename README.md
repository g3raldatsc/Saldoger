# Saldoger

## Apa itu Saldoger

Saldoger merupakan aplikasi web yang saya rancang untuk membantu mencatat pemasukan dan pengeluaran keuangan orang-orang (informasi keuangan) supaya orang-orang yang menggunakan aplikasi ini dimudahkan dalam mengecek informasi keuangan mereka.

---

## Tujuan Pengembangan

Saldoger dikembangkan oleh saya guna membantu orang-orang termasuk saya sendiri, yang merasa sulit mengelola keuangan karena sering sekali susah mengingat informasi keuangan saya. Jadi, aplikasi web Saldoger ini adalah solusi yang terpikirkan oleh saya untuk membantu menyelesaikan masalah tersebut. Disamping hal itu, tujuan saya yang lainnya mengembangkan aplikasi ini adalah sebagai bahan saya untuk berlatih mengelola struktur proyek menggunakan pustaka Django dan Bootstrap, mengelola database, dan menambah pengalaman dalam hal admin interface.

---

## Fitur Utama

Adapun fitur utama dari aplikasi ini adalah:

1. User login.
   Seperti aplikasi pada umumnya, ini membantu sistem menyimpan data user ke database.

2. Fitur Transaksi
   Ini adalah fitur utama untuk user memasukkan data keuangan mereka yang meliputi 
   1. Kategori Transaksi  : Pemasukan dan Pengeluaran
   2. Jenis Transaksi     : 'Informasi keuangannya', misal: uang makan
   3. Waktu               : Waktu memasukkan data atau waktu user mengelola keuangan mereka
   4. Nominal             : Jumlah uang yang masuk/keluar (dalam IDR)

3. Dashboard
   Ini adalah fitur dimana data yang dimasukkan user akan disimpan di dashboard yang berentuk tabel kolom sesuai kategori fitur transaksi.

4. Chart
   Ini adalah fitur dimana ada chart tipe lingkaran donat yang akan menunjukkan visualisasi data user dalam bentuk grafik interaktif.

5. Tampilan 'Saldo saat ini'
   Ini adalah tampilan yang menunjukkan saldo user up to date.
